# HyperCode 🚀

HyperCode is a high-performance, real-time collaborative engineering space that allows distributed development teams to write, edit, and execute code together seamlessly. Powered by WebSockets and the MERN stack, HyperCode eliminates collaboration friction by providing instantaneous code synchronization, active peer tracking, and an integrated deployment execution terminal.

![HyperCode Core Workspace UI Banner](https://images.unsplash.com/photo-1618401471353-b98afee0b2eb?w=1200&auto=format&fit=crop&q=80)

## ✨ Core Features

* **Real-Time Collaborative Editing:** Low-latency typing synchronization across distributed clients using raw WebSocket connections.
* **Persistent Flashlight Glow Interface:** A dynamic, hardware-accelerated mouse-tracking spotlight gradient matching pointer coordinates at 60fps using GSAP.
* **Room Orchestration Sub-System:** Generate unique spatial session instances with cryptographic UUIDs to isolate project workspace data dynamically.
* **Active Peer Matrix:** Live visual head-counters, profile indicators, and sidebar network loops monitoring user online statuses in real-time.
* **Live Execution Console:** Simulated micro-compiler console output mapping local runtime scripts safely directly below the document text area layer.
* **Geometric Layout Architecture:** Glassmorphic navigation components layered systematically above a responsive linear-gradient tech grid background.

## 🛠️ Tech Stack

### Frontend Architecture
* **React (v18+)** - Component UI structures and localized states.
* **Vite** - Lightning-fast building pipeline and development server hot-reloading.
* **GSAP (GreenSock Animation Platform)** - Fluid entrance timelines and cursor-glow tracking engines.
* **Tailwind CSS** - Semantic layout grid system and sleek tech-dark custom utilities.
* **Lucide React** - Clean vector icons.
* **Socket.io-Client** - Duplex persistent networking tunnel pipeline hooks.

### Backend Infrastructure
* **Node.js & Express** - Scalable server-side execution pipeline routers.
* **Socket.io Server** - Event-driven architectural socket orchestration wrapper.
* **UUID** - Cryptographic generation engine for workspace isolation layers.
* **Dotenv** - Multi-environment token masking container setups.
